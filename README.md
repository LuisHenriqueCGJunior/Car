# 🚗 Portfólio Full-Stack — Carros Esportivos

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em desenvolvimento full-stack, utilizando a temática de carros esportivos como base para a criação da interface e experiência do usuário.

A aplicação busca simular um ambiente real de desenvolvimento, envolvendo tanto a construção do front-end quanto a integração com o back-end, focando em organização, escalabilidade e boas práticas.

---

## 🎯 Objetivos

* Evoluir conhecimentos em desenvolvimento front-end e back-end
* Trabalhar com responsividade e layout moderno
* Estruturar melhor projetos reais
* Aplicar boas práticas de código
* Simular um projeto com contexto profissional
